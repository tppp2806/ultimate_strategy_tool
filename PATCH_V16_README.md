# v16：移除无信号选项 + 自动填写蓝色标记

## 改动

1. 移除无信号选项
   - 移除【暂无买点】
   - 移除【暂无破位】
   - 移除【无明显盈利】
   - 将【震荡 / 无趋势】改为【震荡】并保留为大趋势默认项

2. 自动填写标记
   - 脚本自动拉取数据后自动勾选的选项，会显示蓝色边框。
   - 自动标记项右上角会显示“自动”。
   - 用户手动修改该选项后，蓝色自动标记会自动清除。

3. 自动判断为空时
   - 如果入场、破位、盈利、量价辅助未触发，不再勾选任何“无信号”按钮。
   - 自动数据区仍会用 `--` 表示未触发。
