# V39 覆盖文件补丁：回测配置复用左侧配置

把本补丁内文件复制到原项目根目录覆盖即可。

## 覆盖文件

```text
app.py
templates/index.html
static/app.js
PATCH_V39_README.md
```

## 改动

1. 回测页不再重复填写以下字段，统一优先使用左侧【标的与资金】和顶部【设置】：
   - 当前标的 / 市场 / 标的类型
   - 数据源 / 代理 / 超时 / 重试 / 蛋卷 Cookie
   - 计划资金（作为回测初始资金）
   - 仓位模式
   - 策略档位
   - 单笔风险预算 %

2. 回测页只保留真正属于回测本身的参数：
   - 开始日期 / 结束日期
   - 操作周期
   - 最小执行变化
   - 手续费 / 滑点
   - 固定 PE / PB / ROE 假设
   - 是否导出结果表格

3. 切换到【历史回测模拟】时，左侧会隐藏：
   - 当前持仓
   - 当前涨跌幅 / 持仓盈亏 %

4. 后端同步加固：即使请求体里传入了策略档位、仓位模式、风险预算或初始资金，也优先以当前配置中的左侧参数为准，避免页面字段不一致。
