# v20 修复说明：量价无信号降权 + 估值不再单独触发清仓

## 改动

1. `量价辅助确认` 中勾选 `暂无明显量价信号` 后，其他量价选项会自动降低透明度和对比度。
2. 勾选任意具体量价项后，`暂无明显量价信号` 会降低透明度。
3. 修复估值修正误伤：在没有破位、没有止盈、没有明确卖出信号时，PE/ROE 只作为提示和新增仓位刹车，不会单独把当前仓位减到 0%。
4. 买入/加仓/重仓时，估值和 ROE 仍会参与目标仓位计算。
5. 减仓/止盈时，高估可以强化减仓，但不会跌破系统自动底仓。
