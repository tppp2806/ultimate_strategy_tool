# V14：拉取前清空旧自动数据

## 改动

1. 每次点击【拉取所选标的数据】后，先清空旧标的自动填入的字段：
   - 历史 PE 百分位 %
   - 历史 PB 百分位 %
   - ROE %
   - 买入价到止损价距离 %
   - 趋势/入场/退出/盈利阶段自动选择
   - 量价辅助确认勾选项

2. 新数据里某些字段缺失时，不再沿用上一次标的的旧值。

3. 若拉取失败，旧 PE/ROE/PB 会保持清空状态，避免估值数据错乱；用户可手动填写。

4. `setInput` 已改为支持写入空值，用于明确清空字段。
