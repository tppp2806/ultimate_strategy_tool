# 补丁说明：代理模式 + 多链路重试

本补丁用于降低自动拉取行情/估值时的失败率。

## 新增功能

1. 代理模式配置
   - 系统代理：读取 `HTTP_PROXY` / `HTTPS_PROXY` 等环境变量。
   - 自定义代理：可填写 `http://127.0.0.1:7890` 或 `socks5://127.0.0.1:7890`。
   - 不用代理：临时清空代理环境，适合代理反而干扰国内源时使用。

2. 请求容错参数
   - 请求超时秒数：默认 12 秒，可在 3~60 秒之间设置。
   - 单链路重试次数：默认 2 次，最多 5 次。

3. 多链路自动重试
   - 海外/美股：`yfinance` → `Yahoo Chart API` → `Stooq`。
   - 国内/A股/基金：`AKShare` → `Yahoo Chart API` 后缀兜底。
   - 每条链路会按配置重试，失败后自动切换下一条链路。

4. 前端尝试记录
   - 自动数据区会显示“成功链路”和“数据源尝试记录”。
   - 失败时会返回失败原因，仍可手动填写 PE 百分位、ROE、趋势信号。

## 注意

- 自动数据只用于填表和辅助计算，所有字段仍可手动覆盖。
- 不同数据源的复权、基金净值、成交量口径可能不同；用于仓位辅助足够，但不建议当作精确实盘报价。
- 使用 `socks5://` 代理时，`requirements.txt` 已加入 `requests[socks]`。
