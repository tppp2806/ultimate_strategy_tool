# 覆盖补丁说明

如果你已经有旧版项目，按下面结构覆盖即可：

```text
app.py
requirements.txt
templates/index.html
static/app.js
static/style.css
data/config.json（可选，覆盖会重置本地配置）
```

本版保留原来的前端视觉风格：米色背景、三栏布局、卡片、选项聚焦/淡化、悬浮提示、实时计算、本地保存信号状态。

核心变化：

1. 删除旧的“高量第几天”主规则。
2. 改成趋势/动量 + 止损距离定仓位 + 破位退出。
3. 量价项只作为确认和风险修正，不再单独决定加仓/清仓。
4. 新增标的类型：宽基指数、行业ETF、大市值股票、高波动题材。
5. 新增仓位模式：长期底仓+交易仓、纯交易仓。
6. 新增单笔风险预算和止损距离，自动计算风险仓位上限。
