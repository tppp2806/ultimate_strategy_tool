# v49 覆盖文件补丁：PE百分位来源命名修正

## 覆盖文件

把本补丁内文件复制到原项目根目录覆盖：

```text
app.py
templates/index.html
PATCH_V49_README.md
```

## 改动

1. 设置页【PE百分位来源】移除旧的【自动】选项。
2. 【系统自算历史百分位】改名为【乐咕乐股】。
   - 含义：优先通过 AKShare/乐咕历史 PE 序列获取数据，然后脚本本地计算历史 PE 百分位。
3. 【蛋卷估值优先】改名为【蛋卷（雪球）】。
   - 含义：通过蛋卷/雪球相关接口获取历史 PE 曲线，然后脚本本地计算历史 PE 百分位。
4. 旧配置兼容：如果旧配置里保存的是 `auto`，会自动迁移为 `system_calc`，也就是【乐咕乐股】。
5. 只改命名和配置默认值，不改变交易计算逻辑。

## 说明

现在两个选项的含义更直接：

```text
乐咕乐股：使用乐咕历史 PE 序列自算百分位
蛋卷（雪球）：使用蛋卷历史 PE 序列自算百分位
```
