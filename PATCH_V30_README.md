## 改动

1. 买入上限继续保持动态：受策略档位、风险预算、止损距离、PE百分位、ROE影响。
2. 卖出上限从“基本固定”改为“策略基准 + 风险级别动态修正”。
3. 清仓类信号（触发初始止损 / 跌破200日线）不再显示普通卖出上限，指标中显示接近 100%。

## 新的卖出上限逻辑

基础仍来自策略档位：

| 策略 | 基础卖出上限 |
|---|---:|
| 防守 | 55% |
| 均衡 | 45% |
| 进攻 | 35% |

但最终会根据风险动态变化：

| 信号 | 调整方向 |
|---|---|
| 跌破20日线 | 收敛卖出上限，避免小破位卖太多 |
| 突破失败 | 接近策略基准 |
| 跌破50日线 | 放大卖出上限 |
| 跌破200日线 / 触发初始止损 | 允许退出交易仓 |
| 高PE / 低ROE / 长上影 / 未站稳 / 大盘走弱 | 在已有卖出信号下进一步放大卖出上限 |

注意：PE/ROE不会单独触发卖出，只会在已经有破位/止盈/止损信号时强化降仓速度。

## 自测结果

已跑基础逻辑断言，全部通过：

- 高估但无破位：不主动清仓。
- 强多头回踩不破：仍可加仓。
- 纯交易仓触发初始止损：清仓到0%。
- 长期底仓+交易仓触发止损且高估/市场风险：可清到0%。
- 卖出上限随风险递增：跌破20日线 < 突破失败 < 跌破50日线 < 跌破200日线/初始止损。
- 同一破位下，PE极高 + ROE偏弱 + 大盘风险会提高卖出上限。
- 防守 / 均衡 / 进攻的卖出速度仍保持差异。
