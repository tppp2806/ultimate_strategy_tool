# v36 覆盖文件补丁：网页端历史回测模拟

把补丁内文件复制到原项目根目录覆盖即可。

## 覆盖文件

```text
app.py
templates/index.html
static/app.js
static/style.css
PATCH_V36_README.md
```

## 新增功能

1. 顶部新增【历史回测模拟】按钮。
2. 点击后进入同一个 HTML 内的回测页面，不再只依赖命令行脚本。
3. 支持自由设置：
   - 标的代码、市场、标的类型、历史行情源
   - 初始资金
   - 回测开始/结束日期
   - 操作周期：每 N 个交易日检查一次
   - 仓位模式：长期底仓+交易仓 / 纯交易仓
   - 策略档位：防守 / 均衡 / 进攻
   - 单笔风险预算、手续费、滑点、最小执行变化
   - 固定 PE/PB/ROE（可空）
4. 默认只在页面预览，不导出表格。
5. 勾选【生成 CSV 文件】后，才会导出到：

```text
backtest_reports/
```

## 结果预览

页面直接显示：

- 核心指标
- 交易记录
- 导出文件 / 数据源信息

## 回测执行规则

```text
第 t 日收盘后生成信号
第 t+1 日开盘执行交易
```

这样可以避免用未来数据。

## 重要说明

默认不使用逐日历史估值数据，避免把今天的 PE/ROE 回填到过去造成前视偏差。
如果你填写固定 PE/PB/ROE，它会作为整段回测的固定假设参与计算。
