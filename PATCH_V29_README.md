## 本轮发现并修复的问题

v28 测试发现：当风险仓位上限成为主限制时，PE/ROE 只修正理论目标仓位，最终建议加仓幅度可能仍然相同。  
例如 PE 10% 和 PE 95% 都可能因为“单笔风险预算 ÷ 止损距离”限制，最终都显示加仓 +20%。

v29 修复为：

```text
PE/ROE 不仅修正理论目标仓位，也会修正“本次新增仓位上限”。
```

这样高估限制加仓会真实体现在最终结果里。
