# v23 修复说明：初始止损自动触发 + 结果展示精简

## 改动

1. 自动触发【初始止损】
   - 只有在“当前有持仓 + 当前涨跌幅/持仓盈亏 <= -买入价到止损价距离”时，才自动选中【触发初始止损】。
   - 风险仓位上限低于当前仓位，不再等同于触发初始止损。
   - 手动修改当前涨跌幅或止损距离后，后端计算也会自动按同一规则判断。

2. 右侧实时结果展示
   - 【实时计算结果】改为显示本次动作 + 本次仓位变化，例如：
     - 加仓+12%
     - 减仓-30%
     - 止盈-20%
   - 加仓/买入使用绿色；减仓/止盈/清仓使用红色。
   - 大号数字继续显示本次变化量，而不是目标总仓位。

3. 解释区精简
   - 移除独立【解释】折叠区。
   - 原来的解释内容直接显示在大号结果卡片的副标题位置。
   - 【风险 / 限制】仍保留。
