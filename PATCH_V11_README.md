# v11：自动底仓恢复 + 当前涨跌幅替代买入价

## 改动

1. 移除左侧【长期底仓保留 %】输入框
   - 长期底仓不再由用户手填。
   - 系统根据趋势环境、PE/PB百分位、ROE、大盘/板块风险自动计算。
   - 纯交易仓模式下，系统底仓固定为 0%。

2. 结果面板新增/保留【系统底仓保留】
   - 这是破位后最多保留的底仓，不是仓位上限。
   - 计划资金仍然是 100% 上限。

3. 【持仓成本 / 买入价】改为【当前涨跌幅 / 持仓盈亏 %】
   - 例如盈利 12% 填 12，亏损 5% 填 -5。
   - 盈利R倍数 = 当前涨跌幅 ÷ 止损距离。
   - 自动判断：无明显盈利 / 盈利≥1R / 盈利≥2R / 盈利≥3R。

4. 清理旧配置字段
   - 不再保存 `core_floor_pct`。
   - 不再保存 `cost_basis_price`。
   - 不再保存旧版 `asset_type`。
