# v17：恢复无信号选项 + 调整趋势默认项

## 改动

1. 恢复【无信号】选项：
   - 入场信号：新增/恢复【无信号】
   - 减仓 / 清仓信号：新增/恢复【无信号】
   - 盈利阶段：新增/恢复【无信号】

2. 【减仓 / 清仓信号】里的【无信号】与其他信号互斥：
   - 选择【无信号】会取消【跌破20日线 / 突破失败 / 跌破50日线 / 跌破200日线 / 触发初始止损】
   - 选择任意风险信号会取消【无信号】

3. 大趋势环境：
   - 恢复为【震荡/无趋势】
   - 并放在第一项

4. 自动填写标记：
   - 系统自动填写【无信号】时，也会显示蓝色边框和“自动”标记。
