# 鼠标空白处不显示提示 补丁

覆盖路径：
- app.py
- templates/index.html
- strategies/factors/mini_factors.py
- strategies/families/five_dimension_timing.py
- strategies/families/mini_factor_timing.py
- strategies/families/trend_signal_control.py

改动：
1. 保留原有 data-tip 提示图标。
2. 悬浮说明只在鼠标位于右上角 ⓘ 图标区域时显示。
3. 鼠标停在卡片、字段、选项的空白区域时，不再弹出提示。
4. 兼容旧前端脚本：通过捕获阶段拦截旧的整卡片 hover 提示逻辑。

使用：
解压到项目根目录覆盖，重启 Flask，并在浏览器 Ctrl + F5 强制刷新。
