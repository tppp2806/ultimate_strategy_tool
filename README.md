# 趋势风控仓位助手

本地 Flask 网页工具，用于：

- 输入/搜索股票、ETF、基金名称或代码
- 自动拉取国内外行情数据（国内优先 AKShare，海外优先 yfinance）
- 自动计算 MA20 / MA50 / MA200 / ATR14 / 20日量比
- 自动填入趋势状态、止损距离、PE/PB百分位、ROE（能获取则填，不能获取可手动覆盖）
- 输出：观望 / 试仓 / 买入 / 加仓 / 重仓 / 止盈 / 减仓 / 清仓
- 输出精确目标仓位和买卖金额

## 重要原则

- 计划资金就是 100% 上限。
- 不再根据“标的类型”硬性限制最大仓位。
- PE百分位、PB百分位、ROE都使用具体数值输入，计算用连续曲线，不使用硬分段。
- 自动数据只负责填表，任何字段都可以手动覆盖。
- 没有止损距离，不新增交易仓。
- 量价只作为辅助确认，不单独主导买卖。

## 安装

```bash
pip install -r requirements.txt
```

如果国内环境安装 `akshare` 较慢，可以先只装 Flask/yfinance/pandas，海外数据仍可用；国内数据失败时手动填写即可。

## 运行

```bash
python app.py
```

浏览器打开：

```text
http://127.0.0.1:5000
```

## 数据说明

自动数据依赖第三方免费接口，可能因为网络、限频、接口变化而失败。因此工具始终保留手动覆盖：

- 历史 PE 百分位：0~100
- 历史 PB 百分位：0~100，可空
- ROE：具体百分比，可空
- 止损距离：买入价到止损价的百分比

## 仓位模式

- 长期底仓 + 交易仓：破位时保留你设置的底仓比例。
- 纯交易仓：破位后可降到 0%。

仓位模式不再代表“短期/长期”，也不再按大市值/高波动个股限制上限。
