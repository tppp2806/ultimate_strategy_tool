# v18：恢复更具体的“无信号”文案 + 量价辅助无信号互斥

## 改动

1. 把通用【无信号】恢复为更具体的旧名称：
   - 入场信号：`暂无买点`
   - 减仓 / 清仓信号：`暂无破位`
   - 盈利阶段：`无明显盈利`

2. 【量价辅助确认】新增默认项：
   - `暂无明显量价信号`

3. 【量价辅助确认】互斥逻辑：
   - 勾选 `暂无明显量价信号` 时，会自动取消其他量价项；
   - 勾选任意量价项时，会自动取消 `暂无明显量价信号`。

4. 自动填写标记保留：
   - 系统自动填入的默认项或信号项会继续显示蓝色边框和“自动”标签。
