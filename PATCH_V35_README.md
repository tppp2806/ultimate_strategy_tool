## 解决的问题

1. 回测输出文件名改为中文，更容易识别。
2. `权益曲线.csv`、`交易明细.csv` 使用中文列名。
3. 新增 `核心指标.csv`，用“指标 / 数值 / 备注”三列展示。
4. `核心指标.json` 也改为中文键名。
5. 买卖方向从 `BUY / SELL` 改为 `买入 / 卖出`。
6. 报告中的策略档位、仓位模式改为中文。

## 怎么看结果

优先看 `核心指标_*.csv`：

- 策略总收益%：策略最终赚/亏多少。
- 买入持有总收益%：同周期全仓买入不动的收益，用来对比。
- 策略最大回撤%：策略期间最大亏损坑。
- 买入持有最大回撤%：全仓持有的最大亏损坑。
- 年化收益%：折算成年化后的收益。
- 年化波动%：波动大小。
- 夏普比率：收益相对波动的效率。
- 已实现胜率%：只统计卖出时已经落袋的交易盈亏。
- 盈亏因子：已实现盈利总额 / 已实现亏损总额。
- 平均仓位%：策略平均投入了多少仓位。
- 换手率%：交易频率和摩擦成本压力。

然后看 `回测报告_*.md`，它是可读版摘要。

最后看：

- `权益曲线_*.csv`：适合画图，看策略权益和买入持有的走势。
- `交易明细_*.csv`：逐笔检查为什么买、为什么卖。

## 注意

回测报告只能说明历史规则表现，不能保证未来盈利。默认不把当前 PE/ROE 回填到历史，避免前视偏差。
