# v37 覆盖补丁：历史回测模拟改为三栏内切换

## 覆盖方式

把本补丁里的文件复制到原项目根目录，选择覆盖：

```text
templates/index.html
static/app.js
static/style.css
```

## 改动

1. 顶部【历史回测模拟】不再打开独立隐藏页面。
2. 点击后仍保留左侧【标的与资金】栏。
3. 中间列从【信号选择】切换为【回测配置】。
4. 右侧列从【实时计算结果】切换为【回测结果】。
5. 顶部出现【返回仓位助手】按钮，可切回原来的实时仓位计算页面。
6. 【使用当前标的】移到【回测配置】卡片右上角。
7. 移除了旧的独立 `#backtest-page` 结构，避免点击后前端不显示或重复 ID 干扰。
