# V38 覆盖文件补丁：回测配置复用左侧当前标的

覆盖到原项目根目录：

```text
app.py
templates/index.html
static/app.js
```

## 改动

1. 回测配置页移除重复字段：
   - 标的代码
   - 市场
   - 标的类型
   - 历史行情源

2. 历史回测直接使用左侧【当前标的】与顶部【设置】中的数据源配置：
   - `symbol`
   - `market`
   - `asset_kind`
   - `data_source`
   - 代理 / 超时 / 重试 / 蛋卷 Cookie

3. 如果未选择当前标的，点击【开始回测】会提示：

```text
请先在左侧搜索并选择一个当前标的
```

4. 回测初始资金仍保留在回测配置里，用于模拟不同资金规模；如果留空或小于等于0，则自动使用左侧【计划资金】。

## 设计说明

左侧已经承担“当前标的和数据源”的单一入口，回测页只保留真正属于回测场景的参数：

- 初始资金
- 回测日期
- 操作周期
- 仓位模式
- 策略档位
- 风险预算
- 手续费 / 滑点
- 固定估值假设
- 是否导出表格
